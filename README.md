# Nifty 50 Stock Analysis Dashboard

## Project Overview
This project analyzes Nifty 50 stock performance using Python, Streamlit, and Power BI.
It includes data cleaning, analysis, and interactive visualizations.

## Tools Used
- Python
- Pandas
- Streamlit
- Power BI

## Project Files
- app.py – Streamlit dashboard
- clean_nifty50_stock_data.csv – Cleaned dataset
- Nifty50_Stock_Analysis.pbix – Power BI report
- requirements.txt – Python dependencies

## How to Run the Streamlit App
```bash
pip install -r requirements.txt
streamlit run app.py
